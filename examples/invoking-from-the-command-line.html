Invoking from the command line

This plugin can be invoked from the command line this way:

mvn net.nicoulaj.maven.plugins:checksum-maven-plugin:1.6:file -Dfile=the-file-to-process

This will calculate and output the checksum of the-file-to-process for every algorithm supported by the plugin.
